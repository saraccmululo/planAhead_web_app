# PlanAhead Web App

Full-stack task management app that allows users to add, delete, and complete daily tasks with persistent storage and secure authentication.

---

## Features
- User authentication and secure login  
- Add, delete, and mark tasks as complete  
- Persistent task storage using SQLite  
- Responsive design for desktop and mobile  
- Clean, modular React.js front-end components  

---

## Technologies
React.js, Flask, SQLite, Bootstrap, CSS3

---

## Installation
1. Clone the repository:
   git clone https://github.com/saraccmululo/planAhead_web_app.git
   cd PlanAhead
2. Create a virtual environment:
    python3 -m venv env
    source env/bin/activate   # On Windows: env\Scripts\activate
3. Install backend dependencies:
    pip install -r requirements.txt
4. Install frontend dependencies:
    npm install
5. Run the Flask backend server:
    python app.py
6. Start the React frontend:
    npm run dev
    Open your browser at the URL shown in the terminal

Usage
Sign in to manage your tasks
Add new tasks, mark them as complete, or delete them
Responsive interface adapts for desktop and mobile users
